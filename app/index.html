Components

We build your system automatically from the ground-up making sure that each system is clean and up-to-date. All solutions are integrated together to create a complete solution that helps you deliver applications with quality. Below is a list of all the components you'll find on the solution:

Source Code Management

The solution will contain a fully functional GitLab system. This will serve as Git Repository Management were each user will be able to create and manage his own repositories. They will also be able to do Pull requests to other users and take in advance all the nice features of distributed version control.

Project Management

For managing projects one of the best open-source applications for the job is Redmine. A running instance will be freshly installed. Redmine will be configured to be able to show commits of the GitLab repositories and link them with the issues creating a perfect traceability.

Continuous Integration

Continuous Integration Service Jenkins is installed and ready to be used.

Repository Manager

The Repository Manager Nexus OSS is installed and integrated appropriately with Jenkins, so that the latest can do deploys seemlessly.

Code Metrics

SonarQube is an open platform to manage code quality. As such, it covers the 7 axes of code quality: Architecture & Design, Comments, Coding rules, Potential bugs, Complexity, Unit Tests and Duplications. Once Installed and integrated properly with Jenkins, the projects will be analyzed periodically and their metrics will be stored historically, accessible to be used in order to identify metric trends.

Jabox Orchestrator

Jabox is installed together with the rest of the applications. Jabox is an orchestrator that can manage the generation process of the projects so that they are seemlesly integrated to all components.
